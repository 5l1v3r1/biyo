# Biyo

Bio creator for hackers.

## Installation

Add this line to your application's Gemfile:

    gem 'biyo'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install biyo

## Usage

You can use on command line

    $ biyo

    $ How many field in your bio? 2
    $ 1. field key: name
    $ 1. field value: halit
    $ 2. field key: surname
    $ 2. field value: alptekin

    {
      "name": "halit",
      "surname": "alptekin"
    }
